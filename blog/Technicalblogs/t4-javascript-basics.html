Explain the difference between accessing data from arrays and objects.

An array is a data structure that contains values. Each value in the array is accessed to its position within the array. in Javascript you can call specific arrays using numbers or boolean searches. An object is a data structure that contains both data and functions. there are 7 types of data in javascript (Boolean, Null, undefined, number, string and symbol).
