What is grid based design?

Grid based design is the process of boxing every element on a page page, so that each element fits into a vertical and horizontal design. This makes it easier for webdevelopers to lay out a blueprint to what the page will eventually look like

Whats all the hype about responsive webpages?

Today people access the internet on various platforms (phones, tablets, TVs even smart watches). Each one of these devices has different size screens. its imparative that the user experience is not hampered when people access webpages through these devices, therefore responsive webpages rendors the page to load according to the device that it is on.

What is semantic structure?

Semantic structure can be thought of as a blueprint to the webpage, each element represents specific instructions for a computer to render and for us as humans to understand. When building a website, semantic structure helps a web developer build an easy to follow list of instructions so that another web developer can easily see the structure, and make changes where necessary.
